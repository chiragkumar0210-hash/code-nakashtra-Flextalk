## 🔥 Featured Project: FlexTalk

FlexTalk is a wearable glove that translates sign language into speech and text in real-time using sensors, microcontrollers, and AI.

---

## 💡 Problem
Millions of speech-impaired individuals struggle with communication in daily life.

---

## 🚀 Solution
FlexTalk converts hand gestures into understandable voice output and text, enabling seamless communication.

---

## ⚙️ Features
- Real-time gesture recognition  
- Voice output using speaker  
- Bluetooth connectivity (HC-05)  
- Wearable and portable  

---

## 🛠️ Tech Stack
- Arduino Nano  
- MPU6050  
- Flex Sensors  
- DFPlayer Mini  
- Bluetooth Module
